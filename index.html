Понравившиеся элементы

  1. figcaption - элемент подписи иллюстрации представляет собой подпись (заголовок) или легенду, описывающую остальную часть содержимого родительского элемента figure
  2. html - корневой элемент HTML-документа
  3. img - встраивает изображения в HTML-документ с помощью атрибута src, значением которого является адрес встраиваемого изображения
  4. mark - представляет текст, выделенный в справочных целях из-за своей актуальности в определённом контексте. Например, он может быть использован на странице с результатом поиска, в которой выделяется каждый экземпляр искомого слова.
  5. p - параграф в тексте

Единицы измерения цветов

  1. шестнадцатиричный формат - Формат этот содержит три шестнадцатиричных числа, которые задают уровень красного, зеленого и синего цветов соответственно
  2. RGB - формат, при помощи которого задаются цвета CSS - rgb (red, green, blue). Значения чисел в этом формате представляют собой процентное соотношение от 0 до 100%, но наиболее широкое распространение получило представление значения чисел в диапазоне от 0 до 255.
  3. Определение через классическое название - Существует 17 определений цветов. Большинство браузеров поддерживают также 117 так называемых svg-цветов, или x11
  4. HSL - (hue - оттенок, saturate - насыщенность, lightness - светлота). Оттенок определяется на цветовом круге, задается в градусах от 0 до 359, а насыщенность и светлота задаются в процентах от 0 до 100%.
